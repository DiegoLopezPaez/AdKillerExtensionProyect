Este repo representa a la extension del navegador de AdKiller. el service worker tiene las funcionalidades al presionar el boton de la extension, asi como tambien cumple el rol de comunicacion con el offscreen document y con la API del modelo. El offscreen.js es el encargado de la captura del audio.

Para instalar la extension, simplemente es necesario seguir las instrucciones que se encuentran en https://developer.chrome.com/docs/extensions/mv3/getstarted/development-basics/ en la seccion de "Load an unpacked extension". Luego de haber instalado la extension, es necesario iniciar la API del modelo, que se encuentra en este repo: https://github.com/pipecorso79/adKillerAPI. Una vez con el repo local, ir al proyecto e iniciar la API con python app.py en la linea de comandos (instalar dependecias de ser necesario, el repo tiene informacion de las versiones utilizadas).
